# M4 Sprint 1 — Skill Priority Core

## เพิ่ม
- Skill data สำหรับ Swordman, Mage, Archer และ Healer
- Priority Slots P1–P4
- P1 จำกัดเฉพาะอาชีพปัจจุบัน
- ปลดล็อกสกิลตามระดับ Job
- ตรวจสกิลซ้ำและสกิลที่ยังไม่ปลดล็อกฝั่ง Server
- Cooldown แบบ Tick
- Target selection และ Cast Conditions ขั้นพื้นฐาน
- Physical/Magic damage skills และ Healing skills
- Skill Builder UI
- Battle events ชนิด skill_cast และ heal
- Database migration สำหรับ Job skill loadout

## ยังไม่รวมใน Sprint นี้
